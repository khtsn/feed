Cache replaces Overpass in Active Duty pool

Valve have announced that Cache will replace Overpass following the end of the current season of Premier on July 6.

Cache returns to the active map pool after a seven-year hiatus, having been removed in 2019 in favor of Vertigo before a remake by map creator Shawn "FMPONE" Snelling was revealed in 2025.

The community favorite was then bought by Valve after the release of the remake, with Valve eventually officially releasing it in CS2 in April of this year.

As usual, tournament organizers are expected to introduce the map for the new season, with the first tier-one event after Cache's introduction being BLAST Bounty on July 21.

Making way for Cache is Overpass, which only lasted a year in the active map pool before being removed.

The map was played 13 times at the IEM Cologne Major across Stage 3 and playoffs, tied for fourth place alongside Nuke.

The new map pool is:

  • Dust2

  • Mirage

  • Inferno

  • Nuke

  • Ancient

  • Anubis

  • Cache
