AnimGraph2 goes live in latest CS2 update

Valve has pushed the Animgraph 2 update out of beta and into full release after releasing a trial version at the start of April.

The new update has re-authored all third-person animations and, in many cases, adjusted them in response to player feedback. The update also helps reduce CPU and networking costs associated with animation.

The full release follows an earlier update to the first-person animations back in July 2025, when Valve updated animations for "weapon deploy, firing, reload, and inspect" for all weapons.

Map and animation changes headline CS2 update

In addition to helping with performance-related issues, the new animation system aims to smooth in-air crouching transitions and refactor player height on sloped surfaces, which means some grenade lineups on sloped surfaces may have changed.

The full release notes for the update can be found here, or viewed in full below:

All changes from the animgraph_2_beta build are now live.

[ ANIMGRAPH 2 ]

  • Minor adjustments to viewmodel animations.

  • Adjusted general weapon deploy animation logic.

  • Fixed issues with transitioning between knife attacks.

  • Fixed Elites not shooting in third-person.

[ MISC ]

  • Fixed a bug that allowed silently climbing ladders at run speed by sporadically tapping movement keys.

  • Adjusted ground smoothing at locations where sloped ground surfaces transition to flat ground.

  • Fixed held grenades inheriting incorrect scale in some circumstances such as after being dropped and picked up.

  • Fixed a crash at halftime when transitioning from CT to T.
