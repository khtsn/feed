Valve introduce beta update for animations

Valve stated that the Animgraph 2 animation system will reduce the CPU and networking costs of animations.

Valve have released a beta build for Counter-Strike 2, focused on updating the game's animation system to Animgraph 2.

This update follows a late July 2025 update which rolled out Animgraph 2 animations for first-person animations, including reload, inspect, deploy, and firing animations. Today's update now brings freshly redone third-person animations to the beta branch.
