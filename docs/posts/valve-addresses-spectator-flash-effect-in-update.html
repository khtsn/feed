Valve addresses spectator flash effect in update

Valve has released an update in which it has most notably made changes to last week's introduction of the increased flashbang effect from a spectator's point of view.

The original change made it so that spectators would receive the full extent of the flash effect instead of a slightly transparent version, and have the kill feed obscured, making it difficult to follow the action when spectating a flashed player.

Thursday's update introduced a new convar that controls flashbang opacity for remote spectators and adjusted the effect so that it renders underneath the HUD.

ESL FACEIT Group's Marc Winther addressed the update on X, saying the company would test the changes in time for the kick-off of the IEM Cologne Major.

The full changelog looks as follows:

[ COLOGNE 2026 ]

  • Added ability to bookmark stickers in Cologne 2026 Shop.

[ MISC ]

  • Added convar r_spectator_flashbang_opacity to control flashbang opacity for remote spectators.

  • Flashbang effect for remote spectators now renders underneath the HUD.

  • Stability improvements

[ MAPS ]

Ancient

  • Updated clipping to prevent certain boost locations

Sanctum

  • Updated to the latest version from Steam Workshop (Update Notes)
